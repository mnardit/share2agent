# Integration Guides

Step-by-step setup guides for connecting Share2Agent to your platform.

### Automation Platforms

- **[n8n](https://max.nardit.com/share2agent/guides/n8n)** — Webhook node trigger for workflows
- **[Zapier](https://max.nardit.com/share2agent/guides/zapier)** — Webhooks by Zapier trigger for Zaps
- **[Make](https://max.nardit.com/share2agent/guides/make)** — Custom Webhook trigger for scenarios
- **[ActivePieces](https://max.nardit.com/share2agent/guides/activepieces)** — Webhook trigger for flows

### AI Coding Agents

- **[Claude Code](https://max.nardit.com/share2agent/guides/claude-code)** — Terminal agent with tmux delivery
- **[Cursor](https://max.nardit.com/share2agent/guides/cursor)** — AI IDE with integrated terminal
- **[Windsurf](https://max.nardit.com/share2agent/guides/windsurf)** — Codeium AI IDE
- **[GitHub Copilot](https://max.nardit.com/share2agent/guides/github-copilot)** — Copilot in terminal / VS Code
- **[Aider](https://max.nardit.com/share2agent/guides/aider)** — Open source AI pair programming

### Local LLMs

- **[Ollama](https://max.nardit.com/share2agent/guides/ollama)** — Send pages to local models
- **[LM Studio](https://max.nardit.com/share2agent/guides/lm-studio)** — Local LLM with API endpoint

### Custom

- **[Any webhook](https://max.nardit.com/share2agent/guides/generic)** — Python, Node.js, and curl examples
